Compensation Grid Update the percentage increases associated with each level of performance, as well as based on current quartile of the employee. If you do not want to differentiate by quartile or by performance level you can enter equal values in the corresponding cells. These percentages will be updated automatically on the Employee Compensation sheet.
Competencies Revise current competencies and their definitions, employees assigned these competencies will be updated in real time. Additionally, add new competencies as needed.
Competency Weights Change the weighting system utilized in the performance evaluation model for the overall score calculation for all employees. Enter different weights to weight competencies differently, or utilize an unweighted averge by entering static values for each. A change to these values is not recommended unless the change has been announced to all participants in advance.
Employee Competency Assignments Edit or add to the competencies that each employee has as a part of their evaluation process. An employee's competencies cannot be changed if the self or supervisor evaluations are active.
Page Overview Read an overview of all Job Performance Tracker pages, and find a link to each.
Pay Plans Update the values of your pay plans. Employees assigned to these grades will have their min, mid, and max automatically updated.
Performance Threshold Update the JPT score thresholds for evaluations to determine what scores fall into which performance levels. These thresholds are utilized on the Potential x Performance Matrix, as well as within Employee Compensation table and the Compensation Grid.
Reopen Evaluation Reopen mistakenly 'closed' evaluations. This can only be done if the employee and supervisor evaluations are not currently active. If there is any data in the current employee or supervisor evaluation fields, that data will be overwritten.
