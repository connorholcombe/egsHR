How To

Evaluations

Launch Evaluations
  1. Navigate to the evaluation status page. This page can be found in the "Edit & Launch" menu.
  2. Select the check boxes next to each employee's name that you would like to launch an evaluation for.
  3. Select the "Evaluation Status & Email Options" button at the top left of the table.
  4. Activate the self and/ or supervisor evaluation by checking the applicable box. The checkbox on the right will indicate the evaluation will be active, the checkbox on the left will indicate that that field will be updated.
  5. If your account has e-mail functionality enabled, you may also check the applicable email options to send an automated email to each employee notifying them that their evaluation is active.
  6. Select the "update" button to apply these changes.
Close Evaluations This option should only be utilized once the self and supervisor evaluation periods have ended. Closing an evaluation will finalize and submit the information input in the self and supervisor evaluations. Closing an evaluation means that this employee will no longer show up in the "Submission Status" statistics, but will now have their scores included in the "Score Summary" statistics, as well as the "Succession" pages.
  1. Navigate to the evaluation status page. This page can be found in the "Edit & Launch" menu.
  2. Select the check boxes next to each employee's name that you would like to close an evaluation for.
  3. Select the "Evaluation Status & Email Options" button at the top left of the table.
  4. Select the "Close Evaluation" option to close the self and supervisor evaluations for the selected employees. The checkbox on the right will indicate the evaluation will be closed, the checkbox on the left will indicate that that field will be updated.
  5. Select the "update" button to apply these changes.
Reopen Evaluation Reopening an evaluation will only be done if an employee or supervisor need to have access to a previosly closed evaluation. If the employee or supervisor is simply ready to complete their evaluation in the next review period, you will follow the steps in the "launch evaluations" section.
  1. Navigate to the reopen evaluation page. This page can be found in the "System" menu.
  2. This process cannot be completed for employees whose evaluations are currently open, so ensure both the self and supervisor evaluation is inactive.
  3. Select the "edit" button on the right side of the employee's row whose evaluation needs to be reopened.
  4. Check the "reopen evaluation" option. You can also activate either the self-evaluation or the supervisor-evaluation at this time. If both are left inactive, the employee and supervisor will not be able to access the evaluation.
  5. Select the "update" button to apply these changes. Be sure not to apply this process to the same employee multiple times in a row, as that would overwrite the evaluation data with what was submitted in the previous evaluation period.

Employee Information

Add Employee
  1. Navigate to the add employee page. This page can be found in the "Edit & Launch" menu.
  2. Fill in all fields as prompted.
Edit Single Employee's Data
  1. Navigate to the employee data page. This page can be found in the "Edit & Launch" menu.
  2. Select the "edit" button on the right side of the employee's row, or select the "Change Supervisor" button if you would like to change the supervisor assignment.
  3. Enter your edits.
  4. Select the "update" button to apply these changes.
Edit Multiple Employee's Data
  1. Navigate to the employee data page. This page can be found in the "Edit & Launch" menu.
  2. Select the check boxes next to each employee's name that you would like to apply an edit to.
  3. Select the "Edit" button at the top left of the table.
  4. Enter your edits. The checkbox to the left of each field indicates if that field will be edited.
  5. Select the "update" button to apply these changes.
Change an Employee's Email/ Username
  1. Navigate to the accounts page. This page can be found in the "Edit & Launch" menu.
  2. Select the "edit" button on the right side of the employee's row that you would like to edit.
  3. Enter your edits.
  4. Select the "update" button to apply these changes.
Deactivate or Reactivate an Employee's Account
  1. Navigate to the accounts page. This page can be found in the "Edit & Launch" menu.
  2. Select the "edit" button on the right side of the employee's row that you would like to deactivate or reactivate.
  3. Check the box to activate the employee, or uncheck the box to deactivate the employee's account.
  4. Select the "update" button to apply these changes.
Change an Employee's Competencies
  1. Navigate to the employee competency assignments page. This page can be found in the "System" menu.
  2. Select the "select competencies" button on the right side of the employee's row that you would like to edit.
  3. Select the competencies that should be utilized from the dropdown menu for up to 15 competencies.
  4. Select the "update" button to apply these changes.

Review Evaluation Data

See Submission Statistics & Status
  1. Navigate to the submission status page. This page can be found in the "Review" menu.
  2. Use the filters at the top of the page to see an overview of data from a specific group. Note, the filters will only be applied once you select the "Filter" button.
  3. The table below the charts will show individual employee's submission status.
  4. Only employees whose self or supervisor evaluation is active will be a part of this page.
See Score Statistics
  1. Navigate to the score summary page. This page can be found in the "Review" menu.
  2. Use the filters at the top of the page to see an overview of data from a specific group or time period.
  3. The table below the charts will show individual employee scores.
  4. Note, only evaluations that have been closed will have their scores shown here.
Review Evaluations that Have Not Been Closed
  1. Navigate to the active evaluations page. This page can be found in the "Review" menu.
  2. See the employee's overall self-evaluation and supervisor-evaluation scores in the table.
  3. To review the evaluation in more depth, select the "review evaluation" button on the right side of the table for the employee whose evaluation you would like to review.
Review Evaluations that Have Been Closed
  1. Navigate to the review evaluations page. This page can be found in the "Review" menu.
  2. Select the plus button next to the employee's name that you would like to view. This will open up a dropdown of all evaluations submitted for this employee.
  3. To review the evaluation, select the "review evaluation" button on the right side of the table for the evaluation period you would like to review.

System-Wide Settings

Change System-Wide Settings See the list of system-wide changes that can be made below:
  1. To change the percentage increases in employee salaries that would result from each different score grouping, navigate to the compensation grid page. This page can be found in the "System" menu.
  2. To change any of the competencies utilized in JPT, or to add new competencies, navigate to the competencies page. This page can be found in the "System" menu.
  3. To change the weight of different competencies in the calculation of overall scores, navigate to the competency weights page. This page can be found in the "System" menu.

Succession Planning Tools

How to Utilize the "Succession Planning" Tool
  1. Navigate to the succession planning page. This page can be found in the "Succession" menu.
  2. Locate the employee you are planning for. For example, if your IT Manager "James Wilson" has submitted their resignation, you would select "view succession options" next to their name to see some possible internal candidates for promotion into that role.
  3. Utilize the "Comprehensive-Score Succession" model if you would like to find the relevant candidates based on their total performance evaluation score from the most recent period.
  4. Utilize the "Targeted-Score Succession" model if you would like to find the relevant candidates based on their performance evaluation scores from select competencies from the most recent period.
How to Utilize the "Potential x Performance Matrix"
  1. Navigate to the potential x performance matrix page. This page can be found in the "Succession" menu.
  2. Use the filters at the top of the page to more directly target the department or class titles you would like to review.
  3. Locate employees with the best combinations of potential and performance scores from the previous evaluation period. Each box is color coded where green indicates the best scores, and red indicates the worst scores.

Account

  3. Click the "Update" button to apply the change.

  2. Notify the Evergreen Support team that a new Adminsitrator account needs to be made. If there is a specific username that you would like to have used, please let the team know. Please note, this must be a unique username.
